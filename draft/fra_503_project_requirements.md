## Project Description

เป็นระบบ IoT ที่ประกอบด้วย ส่วนต่อไปนี้

- Devices อย่างน้อย 2 ตัว + Communication + Security หากเป็นกลุ่ม 3 คนต้องมี Application ด้วย โดย

- device นั้นจะต้องมีตัวหนึ่งเป็นตัวตรวจจับ และทำการตรวจจับ หลังส่งผลการสั่งการทำงานของอีกตัว (actuate another device) ได้ โดยการสั่งการทำงานอาจจะเป็นแบบอัตโนมัติหรือไม่ก็ได้\
  devices นั้นเป็นการพัฒนาขึ้นเอง หรือเป็นการเลือกใช้ก็ได้ โดยจะต้องแสดงให้เห็น วิธีการพัฒนา และ/หรือ กระบวนการ setting, config ในรายงาน

- เป็นโจทย์ที่มีความต้องการจริง (proposal)

- ทำการศึกษา และวิเคราะห์ระบบที่มีอยู่แล้ว (proposal)

- มีการสื่อสารกันในระบบ IoT นี้

- ระบบ prototype มีความสมบูรณ์ในระดับทดสอบการใช้งานได้ โดยต้องผ่านการทดสอบการใช้งานก่อนจะหยิบถ่ายนำเสนอสรุปสุดท้าย

- มีความฉลาด อาจเป็นระบบตัดสินใจแบบอัตโนมัติแบบ rule base ที่มี parameters มากกว่า 1 ตัว, เป็นระบบตัดสินใจบนพื้นฐานของข้อมูลตรวจจับแบบไม่อัตโนมัติ แต่เป็นแอปให้คนเลือกตัดสินใจ โดยการวิเคราะห์ข้อมูลที่ใช้สนับสนุนการตัดสินใจนั้นไม่ใช่แค่ rule base

- เป็นระบบตัดสินแบบอัตโนมัติ โดยข้อมูลที่ใช้ประกอบการตัดสินใจนั้นจะต้องผ่านกระบวนการที่มากกว่า rule base
